# mkdocs-wiki

[Notifiarr wiki](https://notifiarr.wiki) source.

## Local Build

### macOS

```
brew install python3
python3 -m venv .venv
source .venv/bin/activate
pip3 install mkdocs mkdocs-material mkdocs-material-extensions mkdocs-glightbox
mkdocs serve
```
