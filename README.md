# mkdocs-wiki

[Notifiarr wiki](https://notifiarr.wiki) source.

## Local Build

### macOS

```
brew install python3
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
mkdocs serve
```
